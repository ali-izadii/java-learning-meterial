# Reflection Internals, Dynamic Proxies, Agents & Instrumentation

> **Level:** 🔴 Expert
> **Status:** ✅ Filled — curated 2026-07-06

---

## 🎯 What a senior candidate must know

- **Reflection internals**: `Class` metadata access; historical inflation mechanism (native → generated bytecode accessors after threshold); **JEP 416** — core reflection is now reimplemented *on top of method handles* (JDK 18+); costs — no inlining across untyped call, boxing, access checks; `setAccessible` and how JPMS restricts it (`--add-opens`).
- **JDK dynamic proxies** (`java.lang.reflect.Proxy`): interface-only, `InvocationHandler`, generated `$Proxy0` classes; how Spring uses them for AOP — and falls back to **CGLIB/ByteBuddy subclassing** for classes (consequences: final methods not intercepted, self-invocation bypasses proxy — the classic `@Transactional` trap).
- **Java agents**: `premain` (startup, `-javaagent:`) vs `agentmain` (dynamic attach); `Instrumentation` API — `addTransformer`, `retransformClasses`; what tools are built on this (profilers, APMs like Datadog/New Relic, JRebel).
- **ByteBuddy** as the practical instrumentation layer; ASM underneath.
- Restrictions tightening: JEP 451 (warn on dynamic agent attach), integrity-by-default direction — good currency signal.
- When reflection is the right tool vs `MethodHandle`/`VarHandle` vs codegen at build time (annotation processors).

## ❓ Senior interview questions

1. Why is reflective invocation slower than a direct call? What changed with JEP 416?
2. How does `Proxy.newProxyInstance` work? Why can it only proxy interfaces?
3. Spring `@Transactional` on a method called from another method *of the same bean* doesn't open a transaction — explain the proxy mechanics behind this.
4. JDK proxy vs CGLIB proxy — differences and limitations of each?
5. `premain` vs `agentmain` — how does each get into the JVM, and what can a `ClassFileTransformer` do?
6. How do APM tools instrument your code without you changing it?
7. Why does JPMS break `setAccessible(true)` on JDK internals, and what is `--add-opens` actually doing?
